# Временной ряд — демонстрационная страница

Открыть `index.html` в браузере. Страница позволяет просматривать события на временной шкале и в виде календаря.

**Функции:**
- **Режим временной шкалы:** Зум колесом мыши (с фокусом под курсором, мин. 1 день, макс. ~200 лет), панорамирование перетаскиванием или кнопками
- **Режим календаря:** Просмотр месячного календаря с событиями, навигация по месяцам
- Создание событий по дате с названием
- События отображаются красными точками (в шкале) или цветными блоками (в календаре)

**Архитектура:**
Проект разделён на модули для удобства развития:
- `app.js` — главная логика, управление режимами, обработка событий
- `timeline.js` — логика и рендеринг временной шкалы (зум, панорамирование, SVG)
- `calendar.js` — логика и рендеринг календаря
- `styles.css` — основные стили (layout, формы)
- `timeline.css` — стили для временной шкалы
- `calendar.css` — стили для календаря
- `index.html` — HTML разметка

**Файлы:**
- `index.html` — страница
- `app.js` — главная логика
- `timeline.js` — временная шкала
- `calendar.js` — календарь
- `styles.css`, `timeline.css`, `calendar.css` — стили
